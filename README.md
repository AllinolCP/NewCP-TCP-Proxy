# NewCP-TCP-Proxy

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install the dependencies of the proxy.

```bash
pip install -r requirements.txt
```

## Usage

```
python3.8 bootstrap.py
```
